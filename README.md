# Project Reorganization

The project has been reorganized into two main directories:

- `/app/landing-page`: Contains the static landing page of the application that has HTML and Vanilla JS/CSS files.
- `/app/dashboard`: Contains the React dashboard frontend.
- `/documentation`: Contains project documentation, diagrams, brochures, and supporting materials.

## How to Run

### Landing Page
To view the landing page, open `/app/landing-page/index.html` in your web browser.

### React Dashboard
To run the React application, navigate to the `app/dashboard` folder:
```bash
cd app/dashboard
npm install
npm start
```
